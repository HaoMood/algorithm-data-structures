class HashTable(__builtin__.object)
    We create a hash table by using a list.
 
The collisions are resolved by open addressing, to be more precise, linear
probing.
 
Attributes:
    _slots (list): Store the key-value pairs.

>>> h[54] = 'cat'

>>> h[17] = 'tiger'
>>> h[77] = 'bird'
